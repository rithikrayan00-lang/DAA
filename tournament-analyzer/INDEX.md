DIVIDE & CONQUER TOURNAMENT FINDER

Problem: Find highest-scoring player using D&C pattern

Solution:
  1. DIVIDE   → Split players into left and right halves
  2. CONQUER  → Recursively find winner of each half
  3. COMBINE  → Compare sub-winners, return higher scorer

Complexity:
  ├─ Time:  O(n)     - each player compared once
  ├─ Space: O(log n) - recursion depth
  └─ Work:  7 comparisons for 8 players (n-1)

Example Tree (8 players):
                    [A,B,C,D,E,F,G,H]
                    /               \
              [A,B,C,D]         [E,F,G,H]
              /      \          /      \
         [A,B]  [C,D]      [E,F]  [G,H]
         / \    /  \       / \    /  \
        A   B  C    D     E   F  G    H

Comparisons: 7 (at each level combining results)
