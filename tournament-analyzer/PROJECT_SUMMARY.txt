                         📊 ALGORITHM OVERVIEW
═══════════════════════════════════════════════════════════════════════════════

DIVIDE & CONQUER TOURNAMENT FINDER
──────────────────────────────────

Problem: Find player with highest score using D&C pattern

Solution:
1. DIVIDE    : Split players into two halves
2. CONQUER   : Recursively find winner of each half
3. COMBINE   : Compare sub-winners, return higher scorer

Visual (8 players):
                    [P1,P2,P3,P4,P5,P6,P7,P8]
                    /                        \
              [P1,P2,P3,P4]              [P5,P6,P7,P8]
              /          \              /          \
         [P1,P2]     [P3,P4]        [P5,P6]     [P7,P8]
         /    \      /    \        /    \      /    \
        P1    P2    P3    P4      P5    P6    P7    P8

Complexity:
  Time:  O(n)     - each player compared once
  Space: O(log n) - recursion depth
  
Code (C):
  Player findWinner(Player* arr, int left, int right) {
      if (left == right) return arr[left];
      
      int mid = left + (right - left) / 2;
      Player leftWinner = findWinner(arr, left, mid);
      Player rightWinner = findWinner(arr, mid+1, right);
      
      return (leftWinner.score >= rightWinner.score) ? 
             leftWinner : rightWinner;
  }

Code (JavaScript):
  findWinner(arr, left, right) {
      if (left === right) return arr[left];
      
      const mid = Math.floor(left + (right - left) / 2);
      const leftWinner = this.findWinner(arr, left, mid);
      const rightWinner = this.findWinner(arr, mid + 1, right);
      
      return leftWinner.score >= rightWinner.score ? 
             leftWinner : rightWinner;
  }

❌ Port 8080 already in use

✓ Or change port in api_server.c and rebuild

✓ Check GCC version: gcc --version (should be 4.9+)

✓ See WINDOWS_SETUP.md for detailed solutions

❌ Frontend shows no results
✓ This is expected - JavaScript fallback is working
✓ Check browser console: F12 → Console
✓ Verify backend running: curl http://localhost:8080/api/players

❌ Players not saving
✓ Clear browser storage: F12 → Application → Clear site data

✓ Check localStorage is enabled
